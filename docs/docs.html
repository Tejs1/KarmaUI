Introduction

KarmaUi is a free, open source framework that provides ready-to-use frontend components that you can easily combine to build responsive web interfaces.

Installation Guide

To use KarmaUi in your web interfaces , copy the following link and paste it in tag of your main HTML file. 


    <link rel="stylesheet" href="https://karmaui.netlify.app/min.css" />

Avatar

The Avatar component is used to represent a user, and displays the profile picture, we have avatar in different sizes and shapes. You have to add km-avatar class in img tag to use it as avatar. By default the avatar size is 2.5rem.

Round Avatar Sizes

You have to apply km avatar class to use avatar, which has four different sizes sm | 32px, md | 40px, lg | 64px, xl | 80px ,xxl | 96px. 


    <img class="km avatar sm" src="image-url"/>
    <img class="km avatar" src="image-url" />
    <img class="km avatar lg" src="image-url" />
    <img class="km avatar xl" src="image-url" />
    <img class="km avatar xxl" src="image-url" />

Squared Avatar

Use rs , rm, rl , rf class to add round border to avatar. 


    <img class="km avatar rs" src="image-url" alt="avatar"/>
    <img class="km avatar rm" src="image-url" alt="avatar"/>
    <img class="km avatar rl" src="image-url" alt="avatar"/>
    <img class="km avatar rf" src="image-url" alt="avatar"/>

Border To Avatars

You can use br class with colors to add border 


  <img class="km avatar br" src="image-url" alt="avatar"/>
  <img class="km avatar br black" src="image-url" alt="avatar"/>

  <img class="km avatar br red" src="image-url" alt="avatar"/>

small = 96px, medium = 128px, and large = 192px.

    const ctModalParent = document.querySelector(".km-modal-dialog");
    const modalClose = document.querySelector(".km-modal-close");
    
    modalBtn.addEventListener("click", () => {
      ctModal.classList.add("km-modal-active");
      ctModalParent.classList.add("km-dialog-active");
    });
    
    modalClose.addEventListener("click", () => {
      ctModal.classList.remove("km-modal-active");
      ctModalParent.classList.remove("km-dialog-active");
    });
