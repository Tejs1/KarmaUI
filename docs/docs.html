Introduction

KarmaUi is a free, open source framework that provides ready-to-use frontend components that you can easily combine to build responsive web interfaces.

Installation Guide

To use KarmaUi in your web interfaces , copy the following link and paste it in tag of your main HTML file. 


    <link rel="stylesheet" href="https://karmaui.netlify.app/min.css" />

Avatar

The Avatar component is used to represent a user, and displays the profile picture, we have avatar in different sizes and shapes. You have to add ct-avatar class in img tag to use it as avatar. By default the avatar size is 2.5rem.

Round Avatar Sizes

You have to apply km avatar class to use avatar, which has four different sizes sm | 32px, md | 40px, lg | 64px, xl | 80px ,xxl | 96px. 


    <img class="km avatar sm" src="image-url"/>
    <img class="km avatar" src="image-url" />
    <img class="km avatar lg" src="image-url" />
    <img class="km avatar xl" src="image-url" />
    <img class="km avatar xxl" src="image-url" />

Squared Avatar

Use rs , rm, rl , rf class to add round border to avatar. 


    <img class="km avatar rs" src="image-url" alt="avatar"/>
    <img class="km avatar rm" src="image-url" alt="avatar"/>
    <img class="km avatar rl" src="image-url" alt="avatar"/>
    <img class="km avatar rf" src="image-url" alt="avatar"/>

Border To Avatars

You can use br class with colors to add border 


  <img class="km avatar br" src="image-url" alt="avatar"/>
  <img class="km avatar br black" src="image-url" alt="avatar"/>
  <img class="km avatar br white" src="image-url" alt="avatar"/>
  <img class="km avatar br red" src="image-url" alt="avatar"/>
  <img class="km avatar br green" src="image-url" alt="avatar"/>

Alert

Alerts are use to show a state to user that something affected to a system, feature or page. We have four types of alerts with icons and button, which can you use adding class km-alert in it.

Status Alert

Use km-error for Error alert, km-update for success, km-report for report, km-info for information.

error There was an error processing your request
check_circle Data uploaded to the server. Fire on!
report Seems your account is about expire, upgrade now
lightbulb Mangalyaan is going live on 5 November 2013. Get ready! 

  <div class="km-error km-alert"> 
    <span class="material-icons"> error </span>
    There was an error processing your request 
  </div>
  <div class="km-update km-alert">
    <span class="material-icons"> check_circle </span>
    Data uploaded to the server. Fire on! 
  </div>
  <div class="km-report km-alert"> 
    <span class="material-icons"> report </span>
    Seems your account is about expire, upgrade now 
  </div>
  <div class="km-info km-alert"> 
    <span class="material-icons"> lightbulb </span>
    Mangalyaan is going live on 5 November 2013. Get ready! 
  </div>

Status Alert with Border

Add km-error-bd, km-update-bd... etc classes to add outer border alert. -bd stands for border.

error There was an error processing your request
check_circle Data uploaded to the server. Fire on!
report Seems your account is about expire, upgrade now
lightbulb Mangalyaan is going live on 5 November 2013. Get ready! 

  <div class="km-error-bd km-alert"> <span class="material-icons"> error </span>
  There was an error processing your request </div>
  <div class="km-update-bd km-alert"> <span class="material-icons"> check_circle </span>
  Data uploaded to the server. Fire on! </div>
  <div class="km-report-bd km-alert"> <span class="material-icons"> report </span>
  Seems your account is about expire, upgrade now </div>
  <div class="km-info-bd km-alert"> <span class="material-icons"> lightbulb </span>

Alert with Button

We have close button in this alert type. You can use both default and outer alert types in it using those classes.

error There was an error processing your request close
check_circle There was an error processing your request close 

  <div class="km-error km-alert"> <span class="material-icons"> error </span>
    There was an error processing your request <span class="material-icons"> close </span>
  </div>
  <div class="km-update-bd km-alert"> <span class="material-icons">check_circle </span>
    There was an error processing your request <span class="material-icons"> close </span>
  </div>
